## Requirements

- Python 3
- OpenCV
- Numpy
- Pynput

## Installation

- Create a virtual environment using system packages: `virtualenv -p python3 --system-site-packages venv`
- Use virtualenv with `source venv/bin/activate`
- Install required packages: `pip install -r requirements.txt`
